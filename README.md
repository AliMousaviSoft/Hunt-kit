🐛 Hunt Kit
===========

A structured, evolving knowledge base for **bug bounty hunting**, **web security**, and **offensive methodology**.

This repository acts as a personal *second brain* --- combining cheat sheets, deep-dive notes, and real-world techniques gathered from research and hands-on practice.
